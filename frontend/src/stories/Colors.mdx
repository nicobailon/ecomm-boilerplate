## Usage Guidelines

### Do's
- Use semantic color names (e.g., `text-primary`, `bg-destructive`)
- Rely on the theme system for automatic dark mode support
- Test color combinations for accessibility (4.5:1 contrast ratio)
- Use status colors consistently for their intended purpose

### Don'ts
- Don't hardcode color values (e.g., `text-green-500`)
- Don't mix semantic and hardcoded colors
- Don't use status colors for decoration
- Don't create custom colors without adding them to the theme

## Accessibility

All color combinations meet WCAG AA standards:
- Normal text: 4.5:1 contrast ratio
- Large text: 3:1 contrast ratio
- UI components: 3:1 contrast ratio

Test your color combinations with the Storybook a11y addon to ensure compliance.
