## Typography

Our typography system uses a modular scale to maintain consistent and harmonious text sizes across the application.

### Font Sizes

```css
--font-size-xs: 0.75rem;    /* 12px */
--font-size-sm: 0.875rem;   /* 14px */
--font-size-base: 1rem;     /* 16px */
--font-size-lg: 1.125rem;   /* 18px */
--font-size-xl: 1.25rem;    /* 20px */
--font-size-2xl: 1.5rem;    /* 24px */
--font-size-3xl: 1.875rem;  /* 30px */
--font-size-4xl: 2.25rem;   /* 36px */
```

## Spacing

Our spacing system is based on a 4px unit (0.25rem). This creates a consistent rhythm throughout the interface.

### Spacing Scale

```css
--spacing-1: 0.25rem;   /* 4px */
--spacing-2: 0.5rem;    /* 8px */
--spacing-3: 0.75rem;   /* 12px */
--spacing-4: 1rem;      /* 16px */
--spacing-5: 1.25rem;   /* 20px */
--spacing-6: 1.5rem;    /* 24px */
--spacing-8: 2rem;      /* 32px */
--spacing-10: 2.5rem;   /* 40px */
--spacing-12: 3rem;     /* 48px */
--spacing-16: 4rem;     /* 64px */
```

### Component Spacing

- **Card Padding**: 1.5rem (24px)
- **Button Height**: 2.5rem (40px)
- **Input Height**: 2.5rem (40px)
- **Focus Ring Width**: 2px
- **Focus Ring Offset**: 2px

## Border Radius

Border radius values adapt based on the active theme:

- **Default Theme**: 0.5rem (8px)
- **Modern Theme**: 0rem (sharp corners)
- **Playful Theme**: 1.5rem (24px)

## Animation

### Timing

- **Fast**: 200ms (color transitions)
- **Normal**: 300ms (general transitions)
- **Slow**: 500ms (complex animations)

### Easing

- **Default**: `ease`
- **In-Out**: `ease-in-out`
- **Spring**: Custom spring physics for drawers and modals

## Usage in Code

### Using CSS Variables

```css
.custom-element {
  background: rgb(var(--primary));
  color: rgb(var(--primary-foreground));
  padding: var(--spacing-4);
  border-radius: var(--radius);
}
```

### Using Tailwind Classes

```jsx
<div className="bg-primary text-primary-foreground p-4 rounded-lg">
  Content
</div>
```

### With Opacity

```jsx
<div className="bg-primary/50"> /* 50% opacity */
  Semi-transparent primary background
</div>
```

## Accessibility Considerations

- All color combinations meet WCAG AA standards (4.5:1 for normal text, 3:1 for large text)
- Focus indicators use a 2px ring with proper contrast
- Reduced motion preferences are respected
- High contrast mode adapts borders and outlines

## Theme Customization

To create a custom theme, define all required CSS variables:

```css
.theme-custom {
  --background: 255, 255, 255;
  --foreground: 0, 0, 0;
  /* ... all other required variables */
  --radius: 0.5rem;
}
```

See the [Theme Guide](/docs/ui-primitives-theme--docs) for detailed instructions on creating custom themes.
